# Explore examples

Use the example projects in this repository to bootstrap common setups quickly.

## Example projects

```text
examples/openai-basic   minimal local OpenAI setup
examples/groq-ci        Groq plus GitHub Actions workflow
examples/langchain-app  evalflow alongside a LangChain app
```

Start with the example closest to your stack, copy its `evalflow.yaml` and dataset, then run:

```bash
evalflow doctor
evalflow eval
```

```text
You now have a working quality gate you can adapt to your own prompts.
```
